# Cloudvisio for Node.js

A node module to perform api requests to [Cloudvisio](http://cloudvisio.com)


## Install

Using npm:
```
npm install cloudvisio-api
```


## Usage

Full list of API methods available here: http://cloudvisio.com/docs#api
